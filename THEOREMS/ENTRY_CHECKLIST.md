# Monument Entry Checklist (Alpha)

A theorem entry is Monument-grade only if:

- [ ] Folder is `THEOREMS/NNN-<slug>/`
- [ ] File is `README.md`
- [ ] Title: `Theorem NNN — <name>`
- [ ] Status + License + Authors present
- [ ] Definitions / assumptions explicit
- [ ] Theorem statement explicit
- [ ] Proof included (or clearly labeled “Proof Sketch”)
- [ ] Operational meaning included (what it certifies / how to use)
- [ ] No placeholder words (“PLACEHOLDER”, “TODO”, “lorem”, “fill in”, etc.)
